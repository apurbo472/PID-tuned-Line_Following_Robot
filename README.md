# PID-tuned-Line_Following_Robot
High-speed PID-controlled Line Following Robot built using Arduino Nano, an 8-sensor IR array, and TB6612FNG motor driver.A tuned PID (Proportional–Integral–Derivative) algorithm continuously adjusts motor speeds based on the line position error, enabling smooth tracking, reduced oscillations, and reliable handling of sharp turns at high speeds.


# Fast Line Following Robot

## Overview
A high-speed Line Following Robot built using Arduino Nano and an 8-sensor IR array.

## Features
- PID control
- 8 IR sensors
- Sharp turn handling
- High-speed operation
- TB6612FNG motor driver

## Components
- Arduino Nano
- TB6612FNG
- 8-channel IR sensor array
- Li-ion battery
- DC motors
